Fireworks

    Fireworks （Adobe公司开发的一款软件）由Macromedia（在2005年被Adobe收购）推出的一款网页作图软件，软件可以加速 Web 设计与开发，是一款创建与优化 Web 图像和快速构建网站与 Web 界面原型的理想工具。Fireworks 不仅具备编辑矢量图形与位图图像的灵活性， 还提供了一个预先构建资源的公用库， 并可与 Adobe Photoshop、Adobe Illustrator、Adobe Dreamweaver和Adobe Animate 软件省时集成。 在 Fireworks中将设计迅速转变为模型，或利用来Illustrator、Photoshop和Flash的其它资源。 然后直接置入Dreamweaver中轻松地进行开发与部署。
