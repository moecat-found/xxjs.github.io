答案解析：
1、 B 本题考查的是信息的主要特征。
信息的主要特征有：时效性，共享性，载体依附性，可传递性，不完全性，可伪性，可存储性。
2、 C 本题考查的是计算机硬件系统里常见的输入/输出设备。
键盘、手写笔是输入设备，耳机是输出设备。
常见输入设备：键盘，鼠标，游戏杆，触摸屏，手写笔，写字板，数字转换器，数码相机，数字摄影机，图像扫描仪，传真机，条形码阅读器，字符和标记识别设备，麦克风等
常见输出设备：显示器、打印机、绘图仪、影像输出系统、语音输出系统、磁记录设备等
3、 C 本题考查的是用二进制加法进位规则进行二进制加法计算。本题考试时用计算器：开始—>计算器—>查看—>程序员
二进制是计算技术中广泛采用的一种数制。二进制数是用0和1两个数字来表示的。进位规则是“逢二进一”，借位规则是“借一当二”。对比十进制理解：十进制数是用0~9十个数字来表示的。进位规则是“逢十进一”，借位规则是“借一当十”。
4、 A 本题考查的是常见计算机应用软件。
Core IDRAW是图形处理软件，IOS是苹果公司开发的移动操作系统，Oracle（甲骨文）是关系型数据库管理系统
常见应用软件分类：
文字处理：WORD,WPS，TXT，记事本
音频处理：GoldWave，Mp3ABCut(MP3剪切器)
图形图像：PHOTOSHOP，CorelDRAW，Fireworks，Painter，Illustrator，FreeHand
动画软件：FLASH，万彩动画大师
视频处理：剪辑师，绘声绘影，剪映，Premiere
信息交流：钉钉，腾讯会议，ZOOM，QQ，淘宝旺旺
杀毒软件：360，火绒安全，金山毒霸，瑞星，比特梵德(BitDefender,简称BD,罗马尼亚)，迈克菲（McAfee，麦咖啡，美国），卡巴斯基（Kaspersky，俄罗斯）
程序设计：Python，Scratch，Java，c++，VB
文件搜索工具：everything
5、 C 本题考查的算法描述方式。
算法的描述：自然语言，流程图，伪代码
Python是程序设计语言
机器语言是指计算机硬件能够直接识别的指令的集合,是最早计算机语言，机器语言从属于硬件设备。
6、 B 本题考查的是常见文件格式
MID是音频文件格式、WMV是视频文件格式，DOC是文字处理文件格式
常见文件格式：

文档类：doc(x)，txt，wps，pdf
图形图像：bmp、gif、jpg、pic、png、tif
音频：wav，mid，wma，mp3
视频：AVI，rmvb，rm， mp4，3GP

7、 A 本题考查的是有线网络传输介质
同轴电缆是有线传输介质，微波、激光是无线传输介质。
网络传输介质是网络中传输信息的载体，分为有线和无线两大类。
有线：双绞线，光纤，同轴电缆
无线：微波，激光，红外线，卫星
